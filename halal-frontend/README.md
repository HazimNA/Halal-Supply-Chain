# Halal Supply Chain Frontend

This is the frontend for the Halal Supply Chain project, built with React and Vite.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Configuration

- Ensure MetaMask is installed and connected to Sepolia testnet.
- Update the `CONTRACT_ADDRESS` in `src/blockchain/shared.js` with the deployed contract address from the backend deployment.

## Build

```bash
npm run build
```
